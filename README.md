# Телефонный справочник (База данных жильцов)

Современное десктопное приложение для управления базой данных жильцов с удобным графическим интерфейсом.  
Разработано на Python с использованием Tkinter и PostgreSQL.

![Скриншот приложения](screenshot.png) <!-- placeholder, если добавите скриншот -->

## 🚀 Возможности

- **Просмотр данных** – таблица со всей информацией о жильцах (ID, ФИО, адрес, телефон)
- **Добавление записей** – ввод данных с валидацией полей
- **Редактирование** – изменение существующих записей
- **Удаление** – с каскадной очисткой неиспользуемых значений из справочников
- **Поиск** – по любому полю (регистронезависимый, частичное совпадение)
- **Управление справочными таблицами** – отдельное окно для редактирования списков фамилий, имён, отчеств и улиц
- **Современный интерфейс** – кастомные кнопки с эффектами наведения, тени, адаптивная вёрстка

## 🛠 Технологии

- **Python 3.10+**
- **Tkinter** – графический интерфейс
- **psycopg2** – драйвер PostgreSQL
- **PostgreSQL** – база данных

## 📋 Требования

- Установленный PostgreSQL (версия 12 или выше)
- Python 3.10 или новее
- Операционная система: Windows / Linux / macOS

## ⚙ Установка и запуск

1. **Клонируйте репозиторий**
   ```bash
   git clone https://github.com/your-username/phonebook.git
   cd phonebook
   ```

2. **Установите зависимости**
   ```bash
   pip install psycopg2-binary
   ```

3. **Настройте подключение к БД**  
   Отредактируйте параметры подключения в файле `main.py` (или в начале кода):
   ```python
   DB_HOST = "127.0.0.1"
   DB_NAME = "postgres"
   DB_USER = "postgres"
   DB_PASS = "ваш_пароль"
   ```

4. **Создайте структуру базы данных**  
   Выполните в PostgreSQL следующий скрипт:
   ```sql
   CREATE TABLE fam (f_id SERIAL PRIMARY KEY, f_val VARCHAR(50));
   CREATE TABLE name (n_id SERIAL PRIMARY KEY, n_val VARCHAR(50));
   CREATE TABLE otc (o_id SERIAL PRIMARY KEY, o_val VARCHAR(50));
   CREATE TABLE street (s_id SERIAL PRIMARY KEY, s_val VARCHAR(100));

   CREATE TABLE main (
       uniq_id INTEGER PRIMARY KEY,
       fam INTEGER REFERENCES fam(f_id),
       name INTEGER REFERENCES name(n_id),
       otc INTEGER REFERENCES otc(o_id),
       street INTEGER REFERENCES street(s_id),
       bldn INTEGER,
       bild_k INTEGER,
       ap INTEGER,
       teleph VARCHAR(20)
   );
   ```

5. **Запустите приложение**
   ```bash
   python main.py
   ```

## 🖥 Использование

### Основное окно
- **➕ Добавить** – открывает форму для ввода новой записи (обязательные поля: фамилия, имя, улица, дом)
- **✏️ Изменить** – редактирование выбранной строки
- **🗑️ Удалить** – удаление записи с подтверждением (также удаляет неиспользуемые значения из справочников)
- **🔍 Поиск** – поиск по любому полю
- **🔄 Обновить** – перезагрузка данных из БД
- **📚 Справочники** – управление справочными таблицами (фамилии, имена, отчества, улицы)

### Работа со справочниками
В отдельном окне на вкладках можно:
- Добавлять новые значения
- Редактировать существующие
- Удалять значения (только если они не используются в основной таблице)

## 🗄 Структура базы данных

- **fam** – справочник фамилий
- **name** – справочник имён
- **otc** – справочник отчеств
- **street** – справочник улиц
- **main** – основная таблица с данными жильцов

Все внешние ключи обеспечивают целостность данных.
